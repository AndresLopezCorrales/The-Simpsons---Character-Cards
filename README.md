# The Simpsons - Character Cards 

## The Simpsons API and ThreeJS

Page dedicated to consume the Simpsons API to create cards of the characters that appeared in the TV Show The Simpsons.

For style purposes, ThreeJS was implemented to add a 3D model of Homer Simpson into the page.

## WEB PAGE DEPLOYED

- https://thesimpsonscharactercards.netlify.app/

## Features

- Character cards generated from the Simpsons API

- Pagination for browsing characters

- 3D Homer Simpson model rendered using ThreeJS

- Clean and responsive UI

## Technologies used

- **Javascript (vanilla)**
- **The Simpsons API**
- **ThreeJS**
- **Html && CSS**
- **Tailwind CSS**
